Sends a request to access point 192.168.1.1, if button (D3=DGPIO0) is pressed.  
If an answer is received, the D1mini blue LED blinks.  
IF DEBUG is 1, an answer is sent to Serial (115200 baud)  
All client functions a placed in header file D1_Ex14a_WifiClientButton.h

Hardware:
* WeMos D1 mini
* 1-Button Shield
